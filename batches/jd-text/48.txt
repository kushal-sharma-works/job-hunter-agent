About the Job 🔗
Conference Compass is seeking a skilled and inspired Full Stack Engineer to join our enthusiastic international team. You will help shape the next generation of event technologies that power our highly customizable mobile and web applications for events.
In this role, you will work on the data backbone of our platform: connecting to client systems, importing data from their APIs and static files, and transforming it into the formats our platform needs to deliver rich event experiences. Your work will enable seamless integrations across platforms and services, supporting innovative features for mobile and web applications.
A particularly exciting part of this role is helping us build the next generation of our import tooling, a new suite of data import processes designed to be more automated, more resilient, and AI-friendly. If you're driven by a passion for well-written, efficient code and love tackling complex data and full-stack challenges, we'd love to hear from you. Whether you're an experienced full stack engineer or a rising talent eager to grow, this position offers a fantastic opportunity to make an impact.
What You’ll Do 🧑‍💻
Build and maintain integrations with a wide variety of client APIs, file feeds, and external data sources.
Develop parsing, transformation, and validation logic to map incoming client data into the formats required by the Floq platform.
Design and implement scalable data processing pipelines that reliably handle large volumes of event data.
Build user-facing features in our web applications, owning work end to end from data and API through to the interface.
Help shape and build our next-generation import suite, automated, easier to configure, and AI-friendly by design.
Collaborate with client-facing teams to onboard new data sources quickly and robustly.
Write and maintain code primarily in TypeScript and Python across the stack, with a strong focus on server-side operations.
What We’re Looking For
🤔
We're looking for a full stack engineer with a backend focus who thrives on solving technical challenges and excels in a collaborative team environment. You recognize opportunities, take initiative, and take pride in delivering robust, scalable solutions.
Your Skills and Attributes:
Proficiency in TypeScript and Python for backend development.
Strong understanding of API design and integration (e.g., REST, WebSockets).
Experience building and managing data processing pipelines, including parsing and transforming data from heterogeneous sources (APIs, CSV/JSON/XML files, etc.).
Comfort working across the full stack, including building front-end features (e.g., React).
Familiarity with cloud platforms like AWS, including serverless services (e.g., AWS Lambda, DynamoDB, S3).
An interest in AI-assisted tooling and exploring how LLMs can make data integration work faster and more reliably.
Solid debugging and problem-solving skills with an eye for detail.
Eagerness to learn new tools and solve complex problems.
Fluency in English and excellent communication skills.
